# Источники данных

В этой папке — **откуда брать данные** (файлы, листы, ячейки), в каком формате они приходят и **логика расчётов** по каждому блоку отчётности (ПНЛ, мотивация, касса/банк, остатки, карта и т.д.).

| Документ | Содержание |
|----------|------------|
| [ИСТОЧНИКИ_ДАННЫХ_И_ЛОГИКА_РАСЧЕТОВ.md](ИСТОЧНИКИ_ДАННЫХ_И_ЛОГИКА_РАСЧЕТОВ.md) | Полное описание источников и логики по всем уровням (ПНЛ, мотивация MX, касса/банк, результаты продаж, остатки и карта, ABC, склады, дебиторка, ОС, бухгалтерия, ТП, юристы, сервис). В конце — ревью «что понял / что не понял» и сводная таблица. |
| [СПИСОК_ТРЕБУЕМЫХ_ДАННЫХ.md](СПИСОК_ТРЕБУЕМЫХ_ДАННЫХ.md) | Перечень данных, которые нужны от заказчика (в каком формате, откуда брать). |
| [БАЗА_ДАННЫХ_ОПИСАНИЕ.md](БАЗА_ДАННЫХ_ОПИСАНИЕ.md) | Описание папки «база для данных»: какие файлы там лежат, какие листы и столбцы использовать, связь с Google Drive и формой «Заявка фармацевты». |

При реализации загрузки или расчётов опирайтесь на эти три документа.
